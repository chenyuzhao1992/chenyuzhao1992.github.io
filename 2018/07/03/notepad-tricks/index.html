Reverse List results

Input Filtering

  • int[][] mat
    • if (mat == null || mat.length == 0 || mat[0].length == 0) { return; }
  • String str
    • if (str == null || str.length() == 0) { return; }
  • int[] nums
    • if (nums == null || nums.length < min len of solution) { return; }
  • both null checking (tree)
    • if (a == null && b == null) { return true; }
    • if (a == null ^ / || b == null ) { return false; }

Arrays

  • Arrays.sort()
  • Arrays.asList(nums[i] … )

Duplicates

  • “whether or not / exsiting problems”, no need to de-dup, but “first match, just return”
  • “return all matching solutions” require “de-dup”

Type of Questions

  • Return Boolean
    • Exsiting Problems - not full search, but first match first return
    • Verify - full search
  • Return Solutions
    • All Possible - full search, put them correctly in containers
    • Optimal / Min / Max - full search whne filtering out
