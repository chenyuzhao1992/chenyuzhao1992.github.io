005-longest-palindromic-substring

Austin Zhao - Meaningful & Neat Coding
2018-06-28

DP

  • subproblem is f(i,j) for “substring”, so 2D DP with O(n^2)
  • f(i,j) = s[i] == s[j] && f(i+1, j-1)
    • from here, we know i shoud loop left <- right
    • j should loop left -> right
    • there are 3 cases for this transfer
  • s.substring(i, j+1) cover s[i,j] not s[i,j+1]
